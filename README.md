# upshot-web

A robust and secure source of truth for a varienty of applications at the intersection of DeFi and NFTs.

## Design System link:

```bash
# From the upshot-ui directory
yarn link

# From the upshot-web directory
yarn link @upshot-tech/upshot-ui
```

### Development

```bash
yarn dev
```

### Build

```bash
yarn build
```

### Storybook

```bash
yarn storybook
```

### Formatting

```bash
yarn lint
```
